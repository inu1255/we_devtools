# uniapp android离线打包调试库

实现了io.dcloud.feature.weex.WeexDevtoolImpl, 解决离线打包无法连接到调试服务的问题

## 使用方式

将 `we_devtools-release.aar` 放到android项目如`simpleDemo`的 `libs` 目录下即可
